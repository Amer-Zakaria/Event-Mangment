# SVU Events Directory

A MERN-mentality PHP project for university event management.

## Setup Instructions

1.  **Environment**: 
    - Move project to `htdocs/HW`.
    - Create a `.env` file based on `.env.example`.
2.  **Database**:
    - Create `city_events` database in your MySQL instance.
    - Import `setup.sql` to initialize tables and default admin.
3.  **Run**:
    - Start Apache and MySQL.
    - Access via `http://localhost/HW/index.php`.

## Credentials
- **Admin Login**: `admin` / `admin`
- **Admin Panel**: `http://localhost/HW/admin/login.php`
